# Sparkbook Figma Assets

Source of truth: Figma file `INF2191_A4_Raymond-Zhang--Copy-`, page `Components & Design System`.

Asset folders are reserved for direct Figma exports:

- `logo/`
- `icons/`
- `images/`
- `map/`
- `ui/`

Current implementation notes:

- `icons/figma/` contains direct PNG exports from the Figma component page for the bottom navigation states, add action, and spark map pins.
- The Sparkbook logo is recreated as live text from the Figma prototype wording: `sparkbook ✦` in Main blue `#2E5BAD`.
- No new visual assets, photos, illustrations, icon packs, or generated images were added.
- Profile/avatar treatment is implemented from the Figma initials-based ellipse pattern.
- Category and map markers use the exported Figma spark pin primitive when standalone category-specific icon exports are not present.
